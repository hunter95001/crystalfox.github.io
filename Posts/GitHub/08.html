GitHub 폴더삭제 git rm

GitHub

작성일 : 2019-09-04

git rm "파일명, 폴더명"       <-파일명, 폴더명 삭제
git rm -r "파일명, 폴더명"      <-파일명, 폴더명 삭제
git rm -rf "파일명, 폴더명"     <-파일명, 폴더명 삭제
git rm --cached "파일명, 폴더명"  <-파일명, 폴더명 삭제 , 로컬 저장소 삭제 X
git rm -r --cached "파일명, 폴더명" <-파일명, 폴더명 삭제 , 로컬 저장소 삭제 X
git rm \*            <-전부다 삭제
git rm -r \*           <-전부다 삭제



연습용 레포지토리 입니다.
DB,와 TEST가 있는데 여기서 TEST를 삭제해보도록 합시다

remote 명령어를 사용해서 레포지토리를 연결해줍니다.

적용할 폴더에는 DB와 test폴더가 있습니다

최근에 push를 한 상태이기 때문에 status에는 깨끗한 상태입니다.
git rm test 명령어가 먹히지 않으면
git rm -r test를 통해 삭제해주도록 합시다

status에 삭제되었다고 표기 됩니다.

commit을 적용하여 레포지토리를 변경한 후 push를 해줍니다.
push 후에 삭제된게 적용 됩니다.

삭제가 안된다면


>git status
delete 부분이 빨간색으로 되어있는지 확인해주세요


> git add .
delete 부분을 초록색으로 바꿔주세요
