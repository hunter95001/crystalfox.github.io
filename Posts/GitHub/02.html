cd e:/GitHub [파일이 있는 경로로 이동합니다.]

-메세지 내용은 아래 사진처럼 표기됩니다.[git commit -m "메세지 내용"]

git remote add origin https://github.com/hunter95001/Image.git

remote는 원격 저장소를 연결하는 명령어 입니다 [ git remote add origin "주소"]

git push origin master [일반적으로]

오류 목록

git add. [add . 뛰어쓰고 . 입니다..]

git push origin +master [강제적으로]
