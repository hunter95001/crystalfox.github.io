가상환경 설정


>nvcc -version
-설치된 CUDA정보를 보여줌


>conda --version
-아나콘다 가상환경 버전을 보여줍니다.


>conda update conda
-아나콘다 가상환경의 최신화


>conda create -n "이름"
>conda create -n pytorch
-가상환경을 생성합니다.


>conda info --env
-생성된 가상환경을 확인 합니다.
