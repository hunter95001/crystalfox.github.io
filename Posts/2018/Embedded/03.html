AVR ATmega128 소개

ALU (명령어 2Byte)
=범용 레지스터만 받음
주소 2Byte [ OP | Oprend ]
OP-> 디코드 { ADD }
메모리-> ALU(X)
메모리 -> 범용 레지스터 -> ALU(O);

SR= 상태 레지스터
->가장 최근에 실행된 산술 연산의 결과 정보를 저장 [8Byte]
전역 인터럽트 활성화 비트
-> 부분 인터럽트가 모두 1이더라도 전역 인터럽트가 0이면 인터럽트가 걸리지 않음

스택 포인트 = Push, Pop을 사용 할때 가리키고 있는 것
RAM의 일부를 스택으로 사용 [함수 리턴, 지역 변수 값 등]

플래시 메모리 -비휘발성 메모리
EEPROM -반 영구적으로 저장해야될 값이 있을대
[값이 바뀔수 있는 PARAMETA]
일련의 데이터나 축적된 값을 저장하기 위해 비휘발성 메모리에 저장하기 위함

프로그램 메모리 -PC(2ByTE -> 주소 전달 [16진수] )
FFFE->부트 로드 처음으로 점프 해라
부트로드 크기는 가변
워드 단위로 접근 (2Byte -16)
플레시 메모리 (응용 프로그램 + 부트 프로그램)

범용 레지스터 (32개 =26[Data]+ 6[간접 Addrees])
