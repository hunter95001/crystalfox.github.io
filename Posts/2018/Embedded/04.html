I/O 공간

1. memory -mapped I/O
memory space{
동일한 주소 공간에
RAM안에 I/O공간이 있음
}
*접근에 사용하는 명령어가 같다 (RAM도 MOV , I/O도 MOV)
1. I/O -mapped I/O
memory space + I/O SPACE{
서로다른 독립적인 공간에 있음 [별도로 할당]
}
*접근에 사용하는 명령어가 다름 (RAM도 MOV , I/O도 IN, OUT)
