병렬 통신
직렬 통신
-동기식 =>라인 3개(clock,TX,RX) [clock에 맞춰서 통신 합니다]
-비동기식 =>라인 2개 (TX,RX) [통신하기 전에 통신 parameta를 맞춘다]

비동기식 통신하기전에 약속
1. 전송속도 : 9600BPS//속도가 다르면 깨진다
2. data bit =8bit
3. stop bit =1bit
4. prating bit, no parting


Serial 객체의 메소드
Begin()/end 시작/종료
available() 데이터의 확인 유무
read() /peek() 버퍼의 데이터를 읽음 (버퍼에서 읽고 삭제, 버퍼에서 값만 읽음)
Write / print / println 문자 읽음 /문자열 읽음 /문자열 읽고 띄어쓰기
Flush -out buffer 지움 [송신 buff - 전부다 보내고 지움, 수신 buff- 전부다 지움]
