임베디드 개발과정
1.요구 분석 -> 어떤기능을 구현 하는가
2. Digtal System 구성 결정 -> 구현하기 위한 H.W결정
3. Digtal System 회로 설계 -> 회로 설계, 제작
4. 상태도기반 제어 프로그램 의 설계 제작 -> 알고리즘 설계 프로그램 제작
5. 컴파일, 파일생성
6. 다운로드
7. 디버깅

실행파일 = 실행코드+ 디버깅...등 여러 정보가 내장 되어 있음.
ELF =유닉스,리눅스 실행파일 (.EXE 같은것)
HEX =실행파일에서 실행 코드만 뽑은것.

최적화 설정
-O0 = 최적화 하지 않음 [High Level Languge]
-0,O1 = 기본 최적화
-O2 = -O1보다 더 최적화
[실행속도 + 프로그램 길이를 절충해서 최적화 함]
-O3 = 실행속도 최적화

[함수코드를 프로그램 내부로 포함 => 코드가 길어짐 => 프로그램 용량이 커질수 있다.]
-O4 = 프로그램 크기를 줄인다.
[실행속도 상관 않함]

최적화 설정
*중요{
인터럽트 안에서 일어나는 변수는 Volatile 선언을 해줘야한다.
최적화에 따른 비동기적 문제를 해결하기 위해서 최적화에서 제외해줘야 함.
}
