1. 사이버 보안 빅데이터
1.1 목표설정
- 분석의 최종목표는 무엇인가요
- 어떤 데이터를 활용할 수 있을까요
- 내가 가진 기술은 무엇인가요
- 제약 사항은 없나요
- 결과물을 어떻게 활용할 계획인가요

1.2 데이터 탐색
- 데이터 양은 충분한지 [양질의 데이터는 많으면 많을 수록 좋다] 최소 100만개이상
- 데이터는 분석에 적합한 형태인지
- 데이터에 결함(누락된부분)은 없나요
- 필드 속성에 대한 명세를 확보 할 수 있나요
- 데이터를 분석에 적합한 형태로 변환할 수 있나요

1.3 분석 전략 수립
- 다시, 분석 목표는 무엇인가요
- 신기술 적용이 목표인가요?
- 분석에 투입되는 인원은 몇명인가요?
- 분석 인력의 기술 수준은 어느 정도 인가요?
- 분석을 위한 기술적 환경은 갖춰져 있나요?

1.4 분석/ 모델링
- 언어/ 환경 선택
- 시간
- 삽질
- 시간
- 삽질
- 중요한 것은 결론을 내는 것이 아니라, '좋은 결과물'을 만드는 것

1.5 결과 검증
- 머신러닝의 목표 = 지금껏 한 번도 보지 못한 상황에서도 잘 판단하는 것 !
- 모델링 자체보다 모델 평가 지표와 검증에 힘써야 함
- 단순한 정확도는 이제 버리자 = 99.9% 함정
- 모델의 유형, 분석 도메인 속성에 따라 평가 방법이 상이함
