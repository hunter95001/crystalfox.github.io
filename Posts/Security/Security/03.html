1. 데이터 과학 분야 이해
1.1 데이터 과학 이해하기
데이터 과학(data Science)
- 데이터와 관련된 실제 현상을 이해하고 분석하기 위한 통계학, 데이터 분석 및 머신 러닝 기술을 통합하는 개념
- 수학, 통계학, 정보공학, 패턴 인식, 머신러닝, 시각화,빅데이터 등

데이터 과학 분야에 필요한 기술
- 프로그래밍 능력
- 수학과 통계학
- 도메인 지식/경험

1.2 데이터 과학 성숙도와 목표
데이터 분석 목표
- 현상 파악 : 현재 상황은 어떠한가? [시각화 분석/ 통계 분석]
- 원인 분석 : 문제의 원인, 왜 그렇게 되었는가?
- 미래 예측 : 앞으로 어떻게 달라질 것인가 [모델링]
*과거와 현재의 패턴을 토대로 미래를 예측

데이터 과학의 성숙도 [4,5 단계가 주된 관심사]
1단계[기본] Data collection and basic analysis = 데이터 수집하고 분석
2단계[기본] Data integration and consolidation
3단계[심화] Business reporting and analysis = 데이터에 대한 기술과 심도있는 분석이 필요
4단계[모델링] Predictive analytics =통계 분석과 상관관계를 파악
5단계[모델링] Cognitive analytics = 뉴럴네트크[신경망]기법을 적용해서 인지 분석

현상과 원인을 토대로 앞으로 일어날 일을 대비하고 예방해야함.

1.3 데이터 분석 기술 학습 방법
프로그래밍 능력
- 비전공자 : 최소 3-6개월 정도의 시간 투자
- 전공자 : 데이터 분석 분야에 적합한 언어 + 연습 (1-2개월)

수학과 통계학
- 원로적인 수학도 중요하지만 '코드'와 함게하는 수학을 익히는 것이 가장 좋음

도메인 지식/경험
- 가장 좋은 방법은 해당 분야에 일을 해보는 것!
- 스터디 /학회 / 컨퍼런스 참여

2. 머신러닝과 인공지능
2.1 머신러닝, 딥러닝, 인공지능 개념 잡기
인공지능
- 인간의 학습 능력과 추론 능력을 인공적으로 모델링하여 외부 대상을 지각하는 능력을 컴퓨터 프로그램으로 구현하는 기술

머신러닝 (= 패턴인식 = 데이터 마이닝)
- 컴퓨터가 학습할 수 있도록 하는 알고리즘과 기술을 개발하는 분야

딥러닝
