CREATE


테이블 -> 새로만들기 -> 테이블


열이름, 데이터형식 NULL을 표기해줍니다.


열을 우클릭 하시면 기본키와 제약조건을 설정 해주실수 있습니다.


키모양이 생기면서 NULL허용이 삭제된 것을 보실수가 있습니다 [정상적으로 기본키 설정완료]


예를 누르면 정상적으로 테이블이 생성 됩니다.


테이블명을 적어주시고 확인을 눌러주세요..


1.에 새로고침을 5번 정도해주시면 테이블이 보입니다.
생성된 테이블에 디자인을 클릭하시면 만든 열과 데이터형식을 보여줍니다


생성된 테이블의 쿼리문을 보는법
테이블 -> 테이블 스크립팅 -> CREATE -> 새 쿼리 편집기 창
생성(CREATE)하실때 무조권 삭제(DROP)하고 생성해주는게 좋습니다.
1000개, 10000개식 작업할때 동시에 작업하다보면 중간에 오류가 안보이고생성이 되지 않을 수도 있기 때문입니다.
