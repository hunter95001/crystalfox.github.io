04.MSSQL 데이터베이스 삭제

MSSQL

작성일 : 2019-09-09

데이터베이스 삭제시 사용중 에러 뜰 경우


데이터베이스 우클릭 -> 삭제


확인을 누르면 삭제 됩니다.


사용중일시 삭제할 수 없습니다라고 오류가 발생합니다.


데이터베이스의 속성을 선택합니다.


옵션에서 상태를 찾아줍니다.


엑세스 제한을 MULTI_USERT ->RESTRICTED_USER로 바꿔줍니다.


데이터베이스 속성을 바꾸기 위해서 모든 연결을 끊어주는 문구가 뜨게 됩니다.


모든 연결이 끊기고 데이터베이스는 제한된 사용자가 됩니다.
오른쪽의 쿼리문은 기본형식인
DROP DATABASE 연습용

MS SQL 형식
USE MASTER :
GO
DROP DATABASE 연습용;
GO


이번에는 정상적으로 삭제가 됩니다.
삭제후에 연결을 끊어주셔야 합니다.


새로고침을 하시면 오류가 발생합니다.


재연결 후 정상적으로 데이터베이스가 삭제된 모습입니다.
